== Description ==

#### Partnify Partner Plugin  WordPress

*Adding images to your WordPress site is easy with the Partnify plugin.*

#### Some of the Partnify Partner Plugin for WordPress features

- Connect with your free Partnify account in seconds.
- Well documented. Our [knowledge base](http://partnerhelp.partnify.com/).
- Select for any Partnify vendor, their campaigns and assets (vendors must be on a paid account).
- If you cannot see your vendor pleased advise them to upgrade to a paid account. 

== Installation ==

#### Installing the plugin
1. Ensure your have a Partnify Partner account and can access it. You can create a free account (https://www.partnify.com/invite/). Ensure you select Partner on the signup page.
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Partnify Partner Plugin  WordPress** and click "*Install now*"
2. Alternatively, download the plugin and upload the contents of `partnify.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
3. Activate the plugin
. Add you email address and Set your Partnify API key in the plugin settings.
5. Press connect. The button should go green to indicate you are now connected to Partnify.

#### How to display a display ad in widget areas like the sidebar or footer?
Go to **Appearance > Widgets** and use the **Partnify Partner Plugin for WordPress** widget that comes with the plugin.
